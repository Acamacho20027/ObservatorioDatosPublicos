Formato de Archivos

El Observatorio acepta archivos en dos formatos: CSV (Comma-Separated Values) y JSON (JavaScript Object Notation). Ambos formatos permiten estructurar datos tabulares para su análisis y visualización.

Nota importante: Todos los archivos se procesan localmente en su navegador. Sus datos nunca se envían a nuestros servidores.

table_chart

Estructura CSV

Los archivos CSV deben tener la siguiente estructura:

  • Primera fila: Encabezados de columna separados por comas
  • Filas siguientes: Datos correspondientes a cada columna
  • Valores que contengan comas deben estar entre comillas dobles
  • Codificación: UTF-8 recomendada
anio,provincia,matricula_primaria,tasa_desercion
2021,San José,145000,3.2
2021,Alajuela,82000,3.8
2022,San José,147500,3.0
code

Estructura JSON

Los archivos JSON deben ser un array de objetos, donde cada objeto representa un registro con propiedades clave-valor: 

[
  {
    "anio": 2021,
    "provincia": "San José",
    "hospitales": 12,
    "cobertura_vacunacion": 85.5
  },
  {
    "anio": 2022,
    "provincia": "Alajuela",
    "hospitales": 6,
    "cobertura_vacunacion": 82.3
  }
]

Importante: El archivo JSON debe ser válido. Use herramientas de validación JSON si tiene dudas sobre el formato.

Tipos de Visualizaciones

El sistema soporta tres tipos de gráficos para visualizar sus datos:

bar_chart

Gráfico de Barras

Ideal para comparar categorías. Use columnas categóricas en el eje X y valores numéricos en el eje Y.

show_chart

Gráfico de Líneas

Perfecto para mostrar tendencias a lo largo del tiempo o secuencias ordenadas.

pie_chart

Gráfico de Torta

Útil para mostrar proporciones o distribuciones porcentuales entre categorías.

lightbulb

Ejemplos de Uso

Ejemplo 1: Análisis de Datos Educativos

Cargue un archivo CSV con datos de matrícula por provincia. Configure el eje X con "provincia" y el eje Y con "matricula_primaria". Seleccione gráfico de barras para comparar visualmente las diferentes provincias.

Ejemplo 2: Tendencias de Salud

Cargue un archivo JSON con datos de cobertura de vacunación por año. Use "anio" en el eje X y "cobertura_vacunacion" en el eje Y. Seleccione gráfico de líneas para visualizar la tendencia temporal.

Ejemplo 3: Distribución por Categorías

Para ver la distribución porcentual de registros por provincia, configure "provincia" en el eje X y "Conteo de Registros" en el eje Y. Seleccione gráfico de torta para una representación proporcional.
